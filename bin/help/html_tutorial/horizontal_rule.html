Horizontal Rule Tag

The horizontal rule or <hr> tag is used to draw horizontal lines on a page. It is usually used to divide a page into sections.

It's attributes are:
width
size
align
noshade

Example:
<hr noshade align="center" size="5" width="50%">

"Noshade darkens the line.

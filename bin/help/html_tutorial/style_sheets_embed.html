Embedded Style Sheets

Embedded style sheets are placed in the head section of the HTML page within a set of style tags. If you want them to override linked settings they should be placed after the line of code that accesses the linked style sheet file.

<head>
<link rel="stylesheet" href="http://www.createafreewebsite.net/html40.css" type="text/css">
<style type="text/css">
</style>
</head>

Since embedded style sheets add to the size of your pages they should only be used when standard settings defined by your linked style sheets are not desired.

Example Code:

<head>
<style type="text/css">
body {text-align :center}
p{font-family: Arial, sans-serif ;font-size : 10pt }
h2 {font-family: Arial Black, serif ;font-size : 16pt }
</style>
</head>
